# Flatdango Movie Theater

## Project Description

Welcome to Flatdango Movie Theater! This web application enables users to explore a variety of movies, select their preferred films, and purchase tickets to enjoy the cinematic experience.

## Project Setup

To get started with Flatdango Movie Theater, follow these steps:

1. **Clone the Repository**: Clone this repository to your local machine using the following command:

    ```bash
    git clone https://github.com/your-username/flatdango-movie-theater.git
    ```

2. **Set Up Backend**: This project requires a backend server to provide movie data. You can use a tool like `json-server` to set up a simple backend. Replace `apiEndpoint` in `index.js` with the appropriate backend URL.

3. **Open the App**: Open the `index.html` file in a web browser to begin using the application. You'll be able to browse movies, select one, and purchase tickets.
